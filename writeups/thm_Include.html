Insecure parameter binding

Age bisa dimodifikasi. Sekarang aku mencoba memodifikasi isAdmin untuk mengubah hak akses profil guest. setelah aku ubah muncul menu baru di sisi kanan atas, yaitu API dan setings.

SSRF (Server Side Request Forgery)

Pada settings terdapat form untuk mengisi banner.

Aku mencoba apakah rentan terhadap SSRF dengan memasukan URL dari http server milikku.

PYTHON_SCRIPT
python -m http.server 6000       
Serving HTTP on 0.0.0.0 port 6000 (http://0.0.0.0:6000/) ...
10.49.148.128 - - [10/Mar/2026 21:41:31] "GET / HTTP/1.1" 200 -

form itu ternyata rentan karena bisa melakukan request ke url lain tanpa membatasi request tersebut untuk URL tertentu saja. pada menu API terdapat URL localhost 127.0.0.1 untuk request Internal API dan Get Admins API, sekarang kita coba melakukan request kesana.
